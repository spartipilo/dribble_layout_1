# Dribble Project

Questo progetto è un layout/UI realizzato in Flutter a scopo di studio e pratica, ispirato a un design trovato su Dribbble.

## Obiettivo

L'obiettivo del progetto è replicare un layout moderno per migliorare competenze di:

- Flutter UI
- Widget composition
- Gestione del layout e dello spacing
- Strutturazione dei componenti
- Clean UI development

## Disclaimer

Il design originale non è di mia proprietà.  
Questo progetto è stato sviluppato esclusivamente per esercitazione personale, portfolio e apprendimento, prendendo ispirazione da un layout pubblicato su Dribbble.

Tutti i diritti relativi al design originale appartengono al rispettivo autore.
